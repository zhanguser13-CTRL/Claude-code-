# Claude 宠物伴侣 - 使用指南

## 快速开始

### 方法 1: 双击启动（最简单）

直接双击以下文件之一：
- `start_pet.bat` - 带动画效果的完整版
- `start_pet_simple.bat` - 简化版

### 方法 2: 命令行启动

```bash
# 动画版（推荐）
python claude_pet_companion/animated_pet.py

# 或使用模块方式
python -m claude_pet_companion.animated_pet
```

## 宠物操作指南

### 窗口操作
| 操作 | 说明 |
|------|------|
| 拖拽 | 点击顶部横条或宠物区域拖动 |
| 右键 | 显示操作菜单 |
| 双击 | 与宠物互动（会跳跃） |
| 关闭 | 点击右上角 X 按钮 |
| 最小化 | 点击 - 按钮缩小为小图标 |

### 快捷键
| 按键 | 功能 |
|------|------|
| F | 喂食 |
| P | 玩耍 |
| S | 睡觉/醒来 |
| W | 行走模式 |
| Q | 退出 |

### 状态系统
- **饱食度** 🍖 - 随时间降低，喂食恢复
- **快乐值** 😊 - 玩耍和互动增加
- **能量** ⚡ - 睡觉恢复，玩耍消耗

## 动画效果

动画版宠物包含以下动画：

1. **待机动画** - 轻微呼吸和摇摆
2. **眨眼** - 每3-5秒自动眨眼
3. **进食** - 喂食时的咀嚼动画
4. **行走** - 带跳跃的行走动画
5. **睡眠** - 呼吸动画 + Zzz 飘动
6. **玩耍** - 跳跃 + 星星粒子
7. **粒子效果** - 爱心、星星飘动

## 数据保存

宠物数据自动保存在：
```
C:\Users\<你的用户名>\.claude-pet-companion\
├── pet_state.json          # 宠物状态
└── pet_window_state.json   # 窗口位置
```

## 常见问题

**Q: 宠物不见了怎么办？**
A: 检查任务栏，可能在其他窗口后面。位置会自动记住。

**Q: 如何重置宠物？**
A: 删除 `~/.claude-pet-companion/pet_state.json` 文件

**Q: 宠物不会动？**
A: 确保使用的是 `animated_pet.py` 版本

**Q: 关闭窗口后数据会丢失吗？**
A: 不会，数据会自动保存

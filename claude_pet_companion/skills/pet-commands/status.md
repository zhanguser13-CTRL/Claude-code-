# Pet Status Command

View your pet's current status and stats.

## Usage

```
/pet:status
```

## Display

Shows:
- Name and evolution stage
- Current level and XP progress
- Mood with emoji
- Hunger, Happiness, Energy bars
- Recent achievements
- Coding stats (files, commands, errors)

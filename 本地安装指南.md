# Claude Code Pet Companion - 本地安装指南

## 方式 1: 从源码直接运行（推荐）

### 前提条件
- Python 3.8 或更高版本
- tkinter（通常随 Python 一起安装）

### 运行桌面宠物

```bash
# 进入项目目录
cd H:\宠物\claude-pet-companion

# 运行动画版桌面宠物（带完整动画效果）
python -m claude_pet_companion.animated_pet

# 或运行简化版桌面宠物
python -m claude_pet_companion.simple_window_pet

# 或双击启动脚本
start_pet.bat
start_pet_simple.bat
```

## 方式 2: 本地安装

```bash
# 进入项目目录
cd H:\宠物\claude-pet-companion

# 安装到本地
pip install -e .

# 安装后可以运行
pet-companion status    # 查看宠物状态
pet-companion feed      # 喂食
pet-companion play      # 玩耍
pet-companion sleep     # 睡觉
pet-companion start     # 启动桌面宠物
```

## 宠物操作

### 桌面窗口操作
- **拖拽**: 点击顶部栏或宠物区域拖动
- **右键菜单**: 右键点击显示操作菜单
- **双击**: 与宠物互动
- **快捷键**:
  - `F` - 喂食
  - `P` - 玩耍
  - `S` - 睡觉
  - `W` - 行走
  - `Q` - 退出

### 状态管理
- 宠物状态保存在 `~/.claude-pet-companion/pet_state.json`
- 窗口位置保存在 `~/.claude-pet-companion/pet_window_state.json`

## 功能特点

### 基础功能
- 可拖拽的桌面宠物窗口
- 喂食、玩耍、睡觉互动
- 饥饿、快乐、能量状态系统
- 经验值和等级系统

### 动画版特色
- 呼吸动画
- 眨眼动画
- 行走动画
- 睡眠动画（带 Zzz 效果）
- 进食动画
- 玩耍动画（跳跃+星星粒子）
- 爱心粒子效果
- 自动眨眼

### 最小化模式
- 点击 `-` 按钮最小化为小图标
- 点击小图标恢复完整界面

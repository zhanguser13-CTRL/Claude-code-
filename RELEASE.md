### Installation

```bash

git clone https://github.com/your-repo/claude-pet-companion.git
cd claude-pet-companion

# Install
python install.py
```

### Usage

```
/pet:status  - View pet stats
/pet:feed    - Feed your pet
/pet:play    - Play with your pet
/pet:sleep   - Toggle sleep mode
```

### What's New
- First public release of Claude Code Pet Companion
- Complete pixel art sprite set with 5+ emotions
- Full hook integration for real-time activity tracking
- Achievement system with milestones
- Evolution paths based on coding style
- Meal time bonuses (breakfast, lunch, dinner)

### Known Issues
- Webview panel requires VS Code extension API
- Some hooks may not fire in all Claude Code versions
- State persistence uses local JSON file (not cloud synced)
